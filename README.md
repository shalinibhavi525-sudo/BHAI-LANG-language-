### Hello World

### Variables

### Math Operations
```bhai
bhai x = 10;
bhai y = 5;
bhai bol(jod(x, y));        // 15
bhai bol(ghata(x, y));      // 5
bhai bol(guna(x, y));       // 50
```

### Conditionals
```bhai
agar (score > 90) {
    bhai bol("Waah bhai waah! Top marks!");
} nahi_toh agar (score > 60) {
    bhai bol("Theek hai, pass toh ho gaye");
} nahi_toh {
    bhai bol("Padhai karo, bhai!");
}
```

### Loops
```bhai
// While loop
jab_tak (i < 10) {
    bhai bol(i);
    i = i + 1;
}

// For each loop
har_ek item in [1, 2, 3, 4, 5] {
    bhai bol(item);
}
```

### Functions
```bhai

    bhai bol("Hello, " + naam + " bhai!");
```
